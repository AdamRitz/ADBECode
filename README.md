# ADBECode

Code for **ADBE: Blockchain-Based Authenticated Decentralized Broadcast Encryption for Secure Group Communication**.

This repository contains a prototype implementation of ADBE, including an off-chain Rust implementation and on-chain Solidity contracts for CRS generation, public-key verification, and related blockchain-side operations.

## Overview

Authenticated Decentralized Broadcast Encryption (ADBE) is designed for secure group communication in decentralized environments. The project combines:

- a Rust prototype for cryptographic algorithms, local testing, and performance evaluation
- Solidity contracts for blockchain-assisted CRS generation and verification
- auxiliary code for benchmarking and comparison with related schemes

The goal of this repository is to provide an implementation-oriented companion to the ADBE design and evaluation.
